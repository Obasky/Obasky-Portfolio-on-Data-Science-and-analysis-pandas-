# Python Data Analysis & Jupyter Notebooks Portfolio

Welcome to my data science portfolio repository! This repository contains a collection of data analysis, cleaning, and machine learning projects built entirely using Python and Jupyter Notebooks. It also hosts the cleaned datasets generated throughout these workflows.

## 📁 Repository Structure

*   `notebooks/` - Jupyter Notebooks detailing the step-by-step analysis, visualization, and modeling.
*   `data/` - Cleaned, processed datasets ready for exploration.
*   `requirements.txt` - A list of python packages required to run these projects.
